
# Call Evaluation Criteria

## Overview
This document outlines the key criteria for evaluating calls to ensure quality, customer satisfaction, and effective communication. Each call should be assessed based on the following categories:

## Evaluation Criteria
- **Opening and Preparation**
- **Call Closing**
- **Temp Check**
- **Disposition**
- **Product Knowledge**
- **Attitude**
- **Voice**
- **Mute/Hold**
- **Sales Pitch**
- **Understanding the Customer Issue**
- **Giving the Correct Resolution**
